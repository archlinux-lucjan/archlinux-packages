# Collection of PKGBUILDS for Achlinux:

- freebsd-manpages

- git-git

- krusader-git

- linux-firmware-git

- netbsd-pkgsrc

- pkgconf-git

- sddm-git

# Download:

```
git clone https://github.com/sirlucjan/archlinux-packages.git

```
# Install:

```
cd /some_path/archlinux-plasma-papirus/archlinux-packages/package_name
makepkg -srci

```
